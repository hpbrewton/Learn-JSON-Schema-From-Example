Just a quick update, I got the JavaScript Function to Learner shim all set up and running. Here is an example so we’re on the same page:

We have some function (adapted from r.w. function) that can behave as an oracle by mapping an error thrown to false and an error not thrown to true: 

function(numbers){
    var average = 0;
    var total = 0;
    for (var i = 0 ; i < numbers.length; i++){    
        total += numbers[i];   //add the each number in the array to the cumulative total 
    }
    average = total/i;
    if (isNaN(average)) {
        throw "ahh!"
    }
}

The tool learns the following schema: 

Definitions "2" fromList [("1",AnyOf [~NumberSchema Nothing Nothing]),("2",AnyOf [~ArraySchema (AnyOf [RefSchema "1"]) 1 Nothing])]

With unit percision and recall.
